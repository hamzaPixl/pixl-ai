### Colors to Avoid by Sector

| Sector | Avoid | Reason |
|--------|-------|--------|
| Healthcare | Red as primary | Blood, danger, emergency associations |
| Finance | Neon colors | Undermines seriousness |
| Restaurant | Blue as primary | Suppresses appetite |
| Legal | Bright playful colors | Inappropriate gravity |
| Children | Dark, muted palettes | Too somber |
| Luxury | Oversaturated neons | Cheapens premium perception |
| Non-profit | Gold/luxury tones | Appears wasteful |
| Wellness | Harsh reds, blacks | Contradicts calming expectation |

## Layout Patterns by Sector

Defines the section order and composition per sector type. Each bullet is one page section.

### SaaS / Dev Tools
- Hero centered — headline + subtext + CTA + product screenshot
- Logo bar — trusted-by logos
- Features grid — 3–4 icon cards
- How it works — 3-step process
- Testimonials — quote cards
- Pricing — 2–3 tier cards
- FAQ — accordion
- CTA banner

### AI Products
- Hero fullscreen — bold claim + animated background + CTA
- Live demo or video embed
- Capabilities — bento grid of features
- How it works — animated steps
- Testimonials or case studies
- Pricing or waitlist CTA

### Restaurant / Cafe
- Hero full-bleed — large food photo + overlay text
- About split — story + chef photo
- Menu highlights — dish cards with images
- Gallery — masonry or grid
- Testimonials / reviews
- Hours & location
- Reservation CTA

### E-commerce
- Hero full-width — product showcase + CTA
- Featured products — 4–6 product cards
- Categories — visual navigation
- Benefits bar — shipping, returns, support
- Testimonials / star ratings
- Newsletter signup
- Trust badges

### Corporate / B2B
- Hero split — headline left, graphic right
- Stats bar — 3–4 key metrics
- Services grid — 3–6 service cards
- Case studies — 2–3 client stories
- Process — timeline or numbered steps
- Team — key profiles
- Testimonials — logo + quote
- CTA — consultation or demo request

### Luxury / Premium
- Hero fullscreen — background image/video + minimal text overlay
- Story section — editorial text with large imagery
- Collection / showcase — full-bleed image gallery
- Craftsmanship detail — close-up imagery with captions
- Testimonials — minimal elegant quotes
- Contact — refined form

### Blog / Publication
- Hero editorial — featured article with large image
- Latest posts — 3-column grid of post cards
- Categories — tag navigation
- Newsletter signup
- Popular posts — sidebar or featured section
- About the publication

### Education / EdTech
- Hero centered — value prop + CTA + illustration
- Course cards — visual grid
- How it works — step-by-step
- Stats — students, courses, completion rates
- Testimonials — student success stories
- Instructors — team cards
- FAQ — accordion
- Enrollment CTA

### Developer Tools / Cyberpunk
- Hero dark fullscreen — terminal or code demo + bold headline
- Feature list — dense icon grid
- Code demo / interactive playground
- Integration logos
- Docs or quick-start CTA
- Community or GitHub stats

## Component Variation Matrix

For each layout grid axis, the natural component variants:

| Layout Grid | Hero Style | Card Style | Navigation |
|-------------|------------|------------|------------|
| Centered | Text + CTA centered, visual below | Equal-column grid | Centered links or pill |
| Split-screen | 50/50 text + visual | Horizontal card with image left | Left-aligned bar |
| Asymmetric | Dominant text, offset visual | Variable-size bento tiles | Transparent overlay |
| Bento | Multi-cell feature grid hero | Asymmetric span cards | Floating pill |
| Full-bleed | Edge-to-edge background image | No border, image-first cards | Transparent over hero |
| Broken-grid | Overlapping elements, offset text | Staggered offset grid | Minimal, floating |
